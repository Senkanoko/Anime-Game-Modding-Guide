1. What is mod skin?

-> A mod skin in gaming refers to a custom, unofficial skin (appearance or texture) that modifies the look of in-game characters, weapons, or other assets. These skins are created by the community rather than the game's developers and are often used to personalize the gaming experience.

2. Can I get banned?

-> Yes, using a mod skin can get you banned, depending on the game and how it enforces its rules.

4. How do you install mod skin?

- Step 1: Download XXMI Launcher in the link above(install the MSI one)
- Step 2: Download mod skin and mod fixes(for fix mod) in the link above
- Step 3: Install the MSI file
- Step 4: Once XXMI Launcher is installed, choose the game you want to mod
- Step 5: Select the game you want to mod above the launcher and install tools. Once install tools successfully, press 3 dot button(next to "Start" button) and select "Open Mods Folder"
- Step 6: Open the the mod zip file you have installed in Step 2 and drag the the folder into the Mods folder
- Step 7: Start and enjoy!
* Note: If you have a problem with mod skin, make sure you turn off Dynamic Character Resolution(Genshin) and run the Python mod fix file in the link above. Drag the python file into the mod folder and run it then press F10 in the game and enjoy.
