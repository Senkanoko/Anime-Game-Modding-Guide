Welcome to Senkanoko's Anime Game Modding Webpage!

The webpage shows you how to mod a skin your favorite character in some famous anime game(Genshin, HSR, ZZZ, Wuthering Waves)

I. What is mod skin?

-> A mod skin in gaming refers to a custom, unofficial skin (appearance or texture) that modifies the look of in-game characters, weapons, or other assets. These skins are created by the community rather than the game's developers and are often used to personalize the gaming experience.

II. Can you get banned?

-> Yes, using a mod skin can get you banned, depending on the game and how it enforces its rules.

III. What should you do for safe modding?
  • Always check the game's modding policy before installing mods.
  • Avoid modding online games unless the devs explicitly allow it.
  • Use official modding tools if the game provides them.
  • If unsure, use mods only in offline mode.

IV. What do you need to mod these games?

You don't need a lot of things for modding. Just install the things that are listed below:

  1. XXMI Launcher: For modding skin - Github
  2. A mod skin: Which can be installed in GameBanana

  3. Mod fixes: For fixing broken mod skin
    • Genshin
    • HSR

*Note: Mod fixes can be updated in the future.

V. How do you install mod skin?

-Step 1: Download XXMI Launcher in the link above(install the MSI one)


-Step 2: Download mod skin and mod fixes(for fix mod) in the link above



-Step 3: Install the MSI file


-Step 4: Once XXMI Launcher is installed, choose the game you want to mod


-Step 5: Select the game you want to mod above the launcher and install tools. If game have been installed, the launcher will automatically detected your game file. Once install tools successfully, press 3 dot button(next to "Start" button) and select "Open Mods Folder"


-Step 6: Open the mod zip file you have installed in Step 2 and drag the folder containing Mods into the Mods folder


-Step 7: Start and enjoy!

*Note: If you have a problem with mod skin, turn off Dynamic Character Resolution(Genshin) and run the Python mod fix file in the link above. Drag the Python file into the mod folder and run it then press F10 in the game and enjoy.
